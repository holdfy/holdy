# Banco Saczuck App

Aplicativo Flutter sandbox para simular um banco digital externo ao Gatebox.

## Telas implementadas
- Dashboard/Home
- Adicionar Saldo
- Historico/Extrato
- Pagar (PIX copia e cola, QR Code, Link)
- Configuracoes de Simulacao

## Integracao
O app consome exclusivamente o backend dedicado `banco/backend_banco` via HTTP.

## Rodando
```bash
flutter pub get
flutter run
```
