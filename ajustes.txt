## 1. HOLDFY — Visão do produto

**Conceito:** Marketplace com pagamento protegido (escrow), autenticação, ranking e logística integrada.

### Atores

**Vendedor**
- Cadastra produto
- Tira fotos
- Gera anúncio
- Recebe link de venda
- Envia produto
- Acompanha rastreio
- Recebe pagamento após confirmação
- Responde disputa

**Comprador**
- Recebe link / acessa plataforma
- Faz pagamento (Pix)
- Acompanha pedido
- Confirma recebimento
- Libera pagamento
- Avalia vendedor
- Reclama / devolve
- Abre disputa
- Envia fotos / textos de evidência

**Plataforma (Holdfy)**
- Cria ID da transação
- Segura pagamento (escrow)
- Cria selo de qualidade
- Ranking vendedor
- Ranking comprador
- Compara fotos
- Mantém histórico da conversa
- Julga disputa
- Faz antifraude
- Integra logística
- Libera pagamento

## 2. Módulos do sistema

### 2.1 Pagamento / Escrow

**Funções**
- Receber Pix
- Segurar valor
- Confirmar recebimento
- Liberar para vendedor
- Reembolso automático
- Abrir disputa

**Tecnologias possíveis**
- Pix
- Stellar (cross-border no futuro)
- Ledger de transações
- Wallet interna (opcional)

### 2.2 Autenticação / Trust

**Funções**
- Validação ID
- KYC
- Biometria facial (opcional)
- Verificação documento
- Assinatura digital da transação

### 2.3 Antifraude

**Funções**
- Detectar fraude Pix
- Score de risco
- Device fingerprint
- Análise comportamental
- Match de CPF / nome / telefone
- Blacklist

**IA pode ajudar em**
- Foto falsa
- Conta suspeita
- Golpes recorrentes

### 2.4 Ranking / Reputação

**Vendedor**
- Entregou no prazo
- Reclamações
- Score geral
- Taxa de disputa

**Comprador**
- Reclamações abusivas
- Chargeback-like behavior
- Histórico

**Selo**
- Selo verificado
- Selo premium
- Selo autenticado

### 2.5 Logística

**Integração**
- Correios
- Jadlog
- Melhor Envio
- Transportadoras próprias

**Funções**
- Gerar etiqueta
- Tracking
- Prova de entrega
- Seguro

### 2.6 Disputa (Judge system)

**Fluxo**
1. Comprador reclama
2. Envia foto / vídeo / texto
3. Vendedor responde
4. IA pré-analisa
5. Judge manual ou automático
6. Dinheiro vai para uma das partes
